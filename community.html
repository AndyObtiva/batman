Issues

Found a bug or other nasty in batman.js? It's alpha software, so it's bound to happen! We use GitHub's issue tracker to manage our issues.

Access the issue tracker at http://github.com/Shopify/batman/issues.

Source

batman.js is released as open source software under the MIT license. We invite you to download, modify, and use the code for any purpose the license allows. If you do anything you think other people would find useful, we love pull requests!

The entire project is hosted on GitHub. You can find the repo at http://github.com/Shopify/batman.

GitHub makes collaboration super simple. You can easily fork your own copy of batman.js, and then submit pull requests to have your changes merged back into batman mainline. See the Contributing section of the README for more information.
